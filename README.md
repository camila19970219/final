### 目標：讓更多外國人能更認識台灣、更了解台灣的縣市資訊與分佈 ### 
### 網頁介紹:
1.使用者能點選上面的四大區域按鈕，了解哪些縣市包含在此分類下<br>
2.滑鼠移到各縣市，該縣市會變成黃色，並提供資訊<br>
3.當用戶了解各區域分布後，可以點選下面的縣市位置測驗。地圖會隨機挑選縣市變成黃色，用戶在下面區域輸入縣市名稱，答對會顯示"correct answer",錯誤會顯示“wrong answer”，用戶可以點選下面按鈕決定要不要繼續測驗。<br><p/>
